# DITA XML Sample Project

This repository contains a small sample of topic-based technical documentation written in DITA XML.

## Purpose
The aim of this project is to demonstrate:
- structured authoring using DITA
- separation of content into concept, task, and reference topics
- clear and reusable technical documentation design

## Repository Structure
- `docs/` contains individual DITA topics
- `maps/` contains the DITA map used to organise the topics into a guide

## Sample Topic Types
- **Concept**: explains the background of the issue
- **Task**: provides step-by-step troubleshooting guidance
- **Reference**: provides a quick checklist for review

## Example Use Case
The example content is based on a generic SQL performance troubleshooting scenario after system migration. It is intentionally written as a realistic but non-confidential documentation sample.

## Skills Demonstrated
- Topic-based authoring
- Structured documentation
- Technical communication for system troubleshooting
- Reusable documentation architecture
